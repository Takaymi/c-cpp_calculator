# C/C++ Calculator

Two side-by-side implementations of a command line calculator one in **C** (`c_calculator.c`) and one in **C++** (`cpp_calculator.cpp`) written to compare how the same logic is expressed in each language


## Requirements

- GCC for the C version
- G++ for the C++ version

## Compile & Run

**C:**
```bash
gcc c_calculator.c -o c_calculator
./c_calculator
```

**C++:**
```bash
g++ cpp_calculator.cpp -o cpp_calculator
./cpp_calculator
```
